# FastFood App - Flutter

Aplicación móvil para pedidos de comida rápida (tipo McDonald's/KFC).

## Tecnologías
- Flutter
- Firebase Authentication
- Firestore
- Firebase Storage
- Firebase Cloud Messaging
- Google Maps
- Backend REST (NestJS)

## Módulos
- Autenticación
- Menú digital
- Personalización de productos
- Carrito de compras
- Pago simulado
- Pickup / Delivery
- Tracking de pedidos
- Historial y recompensas
- Recomendaciones

## Instalación
```bash
flutter pub get
flutter run
